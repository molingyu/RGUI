# RGSS3 GUI Framework

一个现代化的适用于 [RGSS3](http://miaowm5.github.io/RMVA-F1/RPGVXAcecn/rgss/index.html) 的 GUI 框架。

## 特性
* 强大的事件处理框架，支持异步事件和`filter`等特性。
* 支持非规则的 UI 组件热区。
* 提供便利的 UI 动画的接口。
* 数据绑定。

## UI组件
- [x] 精灵按钮
- [x] 图片框
- [x] 标签
- [ ] 进度条
- [ ] 文字按钮
- [ ] 面板
